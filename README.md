#  📑 기능 목록

## 문제 1

### input
- pobi, crong의 길이는 2
- [왼쪽 페이지 번호(홀수), 오른쪽 페이지 번호(짝수)]
- 페이지는 1 ~ 400 사이

### output
- pobi가 이기면 1
- crong이 이기면 2
- 무승부 0
- 예외사항 -1

- 왼쪽과 오른쪽이 연속되지 않을 경우
- 숫자가 들어오지 않았을 경우
- 페이지의 숫자가 범위를 벗어났을 경우
